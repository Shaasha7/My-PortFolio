🌐 Shalini Udayakumar - Portfolio Website

A modern, responsive, and interactive personal portfolio showcasing my journey as a Full Stack Developer, my projects, certifications, blogs, technical skills, and coding achievements.

📖 About

This portfolio is designed to represent my skills, experience, and passion for software development. It highlights my academic journey, personal projects, technical expertise, certifications, blogs, and coding profiles through a clean and premium user interface.

The website is fully responsive, supports Light & Dark Mode, and provides a smooth user experience with elegant animations and modern design principles.
